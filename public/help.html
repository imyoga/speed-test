How to Use Terminal Speed Test

Terminal Speed Test is a simple, terminal-style interface for testing your internet connection speed.

Available Commands

  • speedtest - Run a full speed test (download, upload, latency, jitter)
  • speedtest --download - Test download speed only
  • speedtest --upload - Test upload speed only
  • clear or cls - Clear the terminal screen
  • history - View previously entered commands
  • about - View information about the application
  • help - Display available commands

Understanding Your Results

After running a speed test, you'll see these metrics:

  • Download Speed: How quickly data can be retrieved from the internet (in Mbps)
  • Upload Speed: How quickly data can be sent to the internet (in Mbps)
  • Latency: The time it takes for data to travel to its destination and back (in ms)
  • Jitter: The variation in latency over time (in ms)
  • IP Address: Your public IP address
  • ISP: Your Internet Service Provider
  • Server Location: Location of the test server

Tips for Accurate Results

  • Close other bandwidth-intensive applications before testing
  • For Wi-Fi connections, position yourself close to your router
  • Run multiple tests at different times of day for a better overall picture
